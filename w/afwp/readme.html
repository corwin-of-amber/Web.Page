Effectively-Propositional Reasoning About Reachability
Using the Tool

System Requirements

  • Z3 SMT Solver from MSR; the executable, z3 (z3.exe on Windows), must be present on the system path.
  • Python 2.7.x, except for the Windows executable release where Python is already bundled.
  • For GUI:
    • PyQt4
    • pygraphviz
    Without these the tool will still run and provide textual feedback. Notice that currently (January 2013) there is no official version of pygraphviz running on Windows.

Installation Instructions

  1. First, make sure that Z3 can run from the console. Open a terminal (Windows: +R, cmd ⏎; Mac: Applications ▸ Utilities ▸ Terminal) and type the following command: z3 -h. You should see a list of Z3 command-line options.
  2. Extract the files from the downloaded archive matching your operating system.
  3. To check your installation:

    At the terminal command prompt, cd to the directory with the extracted files.

    • For the Windows executable release: run analyze examples\sll-reverse.imp
    • For Python source releases, run python analyze.py examples/sll-reverse.imp You may need to prefix python with a path to the Python executable if it is not found on your system path, e.g. C:\Python27\python.

    You can then try other examples from the examples/ directory.
